Graceful degradation and service workers are related because service workers are a form of graceful degradation. Service workers have the ability to keep apps working even when the user is offline, which would be considered the "max technology" that graceful degradation starts at. Service workers then addresses lower levels with grace by falling back to regular network requests when certain browsers don't support service workers.
